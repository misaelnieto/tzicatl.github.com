Acordeón de Emacs — Por Noe Nieto

Este acordeón lo hago basado en el teclado de una PC de 2011 con Linux y emacs 23.2.1 (emacs-x). Siempre que se pueda, nada de shortcuts raros.

Edición de texto

Propósito Comando emacs Combinación de teclas
Copiar Texto kill-ring-save Alt+W
Cortar Texto kill-region Ctrl+W
Pegar Texto yank Ctrl+Y
Deshacer undo Ctrl+X U

Navegación de ventanas y buffers

Propósito Comando emacs Combinación de teclas
Ocupar toda la ventana Ctrl+X 1
Dividir verticalmente split-window-vertically Ctrl+X 2
Dividir horizontalmente split-window-horizontally Ctrl+X 3
Cerrar división Ctrl+X 0
Moverse a otra división Ctrl+X O
